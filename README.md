# Data-analysis-Project
# Customer Shopping Behavior Analysis

This project analyzes retail transaction data to understand customer shopping behavior and support data-driven business decisions.  
It follows a complete analytics workflow from data cleaning to visualization.

---

## Objectives
- Analyze customer purchasing patterns
- Identify high-value customers and products
- Evaluate the impact of discounts and subscriptions
- Generate actionable business insights

---

## Tools & Technologies
- **Python** – data cleaning and exploratory analysis  
- **PostgreSQL** – analytical SQL queries  
- **Power BI** – interactive dashboards  

---

## Project Structure
```text
data/          # Raw and cleaned datasets
notebooks/     # Python analysis and preprocessing
sql/           # SQL queries
powerbi/       # Power BI dashboard
README.md
```

---

## Workflow
1. Clean and prepare transactional data using Python  
2. Load data into PostgreSQL  
3. Perform business analysis with SQL  
4. Visualize insights in Power BI  

---

## Key Outcomes
- Customer segmentation based on purchase behavior  
- Identification of top-performing products  
- Insights into discount effectiveness  
- Comparison between subscribers and non-subscribers  

---

## Business Impact
This project demonstrates how transactional data can be transformed into actionable insights to support marketing, retention, and pricing strategies.
